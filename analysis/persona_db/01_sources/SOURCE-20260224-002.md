# Source: Using crowdsourced medicine to manage uncertainty on Reddit (Thompson et al., 2022)

## Summary
Analyzed 31,892 posts in r/covidlonghaulers to characterize how people communicate to manage uncertainty in a poorly understood condition.

## Evidence Snippet
> "Findings provide a succinct, yet robust set of themes reflecting the information-seeking (i.e., \"This is happening to me\") and support-seeking functions of long-haulers' talk (i.e., \"Is this happening to you?\")."

## Usage Notes
- Use for DO (online information seeking), SAY (language about uncertainty/anxiety), and VALUE (need for validation).
