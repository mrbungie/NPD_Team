# Claim: Purchasers distrust vendor-built ROI analyses

## Statement
Purchasers report skepticism about vendor-produced ROI analyses and limited ability to validate vendor assumptions internally.

## Evidence Basis
- PHTI playbook includes purchaser quote: "Most of the ROI analyses we get are built by the vendors themselves, using assumptions that make them look good. There's no way we can validate those claims internally." (Source: SOURCE-20260224-009)

## Reasoning
This directly informs trust requirements: transparency, data access, and auditability are must-haves.
