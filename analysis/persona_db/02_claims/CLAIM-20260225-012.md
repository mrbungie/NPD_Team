# Claim: FSE 2.0 aims for nationwide interoperability and standardization

## Statement
Italy's FSE 2.0 program targets nationwide diffusion/omogeneita/accessibilita and requires interoperability across regional fascicoli, with central components (Gateway and EDS) to validate standards and expose data services.

## Evidence Basis
- Developers Italia describes nationwide objectives and explicitly calls out interoperability between regional fascicoli and the Gateway/EDS architecture. (Source: SOURCE-20260225-012)

## Reasoning
This supports Italy-first assumptions about integration constraints and why "works across Regions" is a key adoption requirement.
