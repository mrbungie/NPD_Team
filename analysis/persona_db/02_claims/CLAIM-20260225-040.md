# Claim: Cross-Region scale path is the pilot's core value

## Statement
Pilot-forward owners prioritize pilots with a credible path to scale across Regions; pilots perceived as one-off local integrations trigger leadership veto.

## Evidence Basis
- "I need a pilot that can scale across two Regions without re-building integration from scratch." (Source: SOURCE-20260225-027)
- "Hidden veto: when the pilot looks like a one-off with no path to scale, leadership says no." (Source: SOURCE-20260225-027)

## Reasoning
This defines success criteria beyond local pilot metrics: repeatable deployment and standards-aligned integration.
