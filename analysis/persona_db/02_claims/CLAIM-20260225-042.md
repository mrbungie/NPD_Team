# Claim: Pilot success = measurable signal + predictable scalable integration

## Statement
Successful pilots require a measurable signal plus a scalable, predictable integration path; non-negotiables include privacy compliance, testability, and minimal operational disruption.

## Evidence Basis
- "The pilot must produce a measurable signal and an integration path that scales." (Source: SOURCE-20260225-027)
- "Non-negotiable: privacy compliance plus predictable integration behavior; we cannot create manual workarounds." (Source: SOURCE-20260225-027)
- "Success metrics: pilot signal + integration readiness + minimal operational disruption." (Source: SOURCE-20260225-027)
- "Non-negotiable is predictable operations: no hidden workload and compliant data processing." (Source: SOURCE-20260225-028)

## Reasoning
This consolidates how pilots are judged across functions: measurement, integration predictability, and operational impact.
