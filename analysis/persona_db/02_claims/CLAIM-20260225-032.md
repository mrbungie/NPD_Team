# Claim: Appraisal-ready evidence dossiers are the trust threshold

## Statement
Insurer-side evaluators trust pilots, real-world evidence, and peer-reviewed results most when they can be mapped into an appraisal-ready evidence dossier; vague outcomes framing is treated as untrustworthy.

## Evidence Basis
- "Untrustworthy is when the vendor can't map outcomes to an appraisal-ready dossier..." (Source: SOURCE-20260225-023)
- "Confident when the dossier is appraisal-ready... Anxious when it's a black box with generic outcomes." (Source: SOURCE-20260225-023)

## Reasoning
This is a product requirement for how value is packaged: evidence artifacts must be dossier-ready, not marketing narratives.
