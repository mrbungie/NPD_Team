# Source: Only the anxious ones? (Wetzel et al., 2024)

## Summary
Exploratory cross-sectional survey (869 eligible; matched analysis n=134) examining predictors of symptom checker application use.

## Evidence Snippet
> "Hypochondria has emerged as a significant predictor of SCA use with a consistently stable effect..."

## Usage Notes
- Use for DO: who uses symptom checkers and why.
- Use for risks: symptom checkers may unsettle anxious users (triage + unlikely serious diagnoses).
