# Claim: Symptom checkers have unmet needs around personalization

## Statement
Users report unmet needs in symptom checker apps, including insufficient personalization and inability to express nuanced symptom descriptions.

## Evidence Basis
- Qualitative analysis lists unmet needs including "a lack of personalization" and requests for more complex symptom input for nuance. (Source: SOURCE-20260224-004)

## Reasoning
This informs VALUE by identifying what current tools fail to provide; these become candidates for Kano basics/performance features.
